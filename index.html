My Retroactive Life Journal

...what? Basically one day I was thinking about how there are so many interesting memories I have that only pop into my mind on a whim to be forgotten in a couple minutes. I wished I had written everything down in a journal, so I decided that now I'm going to write down my memories when they come to me until there are no more gaps to fill. I also decided that I didn't like writing by hand, that I want to be able to write anywhere at any time, and that I want my vault of memories to be cool-looking, so I chose to start building a website.
